WooCommerce ile sitenizdeki ürünlerin Paraşüt ile senkronize olmasını aynı zamanda stok ve vergi gibi ayarlarınızı aktarmanızı sağlar.

== Description ==
TemaNinja tarafından hazırlanan Paraşüt Woo Lite eklentisi sayesinde ürünlerinizi zahmetsizce Paraşüt hesabınızla eşleştirebilirsiniz. Ürünlerinizin vergi ve varsa stok takibi özellikleri de Paraşüt hesabınıza yansıyacaktır.

Eğer siparişlerinizi de Paraşüt üzerine aktarmak ve eFatura takibini paneliniz üzerinden yapmak isterseniz sitemiz üzerinden eklentinin ücretli versiyonunu satın alabilirsiniz.

== Installation ==

Eklentinin kurulumu için şu adımları izlemelisiniz


1. `parasut-woo-lite.zip` dosyasını `/wp-content/plugins/` dizinine yükleyin
2. Wordpress admin panelinizdeki 'Eklentiler' menusünden eklentiyi aktive edin
3. `Paraşüt Woo` menüsüne giriş yapın
4. `Paraşüt Woo >> API Ayarları` menüsünden api ayarlarınızı yapın, eğer api ayarlarınızı bilmiyorsanız destek@parasut.com ile iletişime geçerek bu bilgileri talep edin.
5. API bilgilerinizi girdikten sonra siteniz artık Paraşüt üzerinde kayıtlı kategorileri okuyacaktır, bu yüzden ilk olarak yapmanız gereken `Ürünler >> Kategoriler` alanına girerek tüm kategorileri Paraşüt ile eşleştirmek olmalıdır.
6. Kategori eşleştirmeleri bittiğinde ürünlerinizi Paraşüt ile eşleştirmek için `Paraşüt Woo >> Araçlar` alanını ziyaret edebilir ve ürünlerinizi eşleştirebilirsiniz.


== Frequently Asked Questions ==

= API Bilgilerimi nasıl alabilirim? =

Paraşüt ekibiyle iletişime geçerek API bilgilerinizi talep etmeniz yeterlidir.

= Ürünlerin stoklarını takip eder mi? =

Evet bu takibi yapar ancak bu takibi yapabilmesi için önemli olan ürünün stok takibinin etkinleştirilmiş olmasıdır. 

= Vergi oranlarını destekler mi? =

Evet, ürün eğer vergilendirilmişse bu vergi oranını da yansıtacaktır. Aynı zamanda Vergiler Dahil/Hariç ayırımını da yapacaktır.
